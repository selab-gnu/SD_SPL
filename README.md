# _"Jung, P., Lee, S., & Lee, U. (2024). Automated code-based test case reuse for software product line testing. Information and Software Technology, 166, 107372."_

## Abstract
This paper tackles the challenge of efficiently testing a software product line (SPL), where a family of related products share significant portions of code. The authors propose an automated, code‑based test case reuse approach that leverages existing test cases and source code to avoid redundant testing across product variants. The method analyzes code similarity and test case coverage to determine which test cases can be reused for different product variants without compromising fault detection. Empirical evaluation using multiple open‑source SPLs demonstrates that the approach effectively reduces test effort and cost, while maintaining strong fault‑detection capability.
